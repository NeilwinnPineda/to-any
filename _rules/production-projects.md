# Production Projects

Production project generation is paused during the component-contract foundation phase.

Do not add sample production projects, route catalogs, or generated app outputs until the component expo proves the reusable UI contract layer.

Current active work:

- define framework-neutral component contracts in `project-to-any/templates/frontend/components/registry.json`
- prove those contracts in `_templates/angular/src/app/pages/components/`
- keep visual behavior in SuperCSS

When production projects return, they must consume the proven component contracts instead of inventing page-specific UI.
