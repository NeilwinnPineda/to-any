SuperCSS is a token-driven CSS design system. Every component consumes --sc-* custom properties so themes, palettes, and density scales mutate the visual output without touching markup.
Set data-theme, data-palette, data-mode, or data-density on the html element. The mixer layer rewrites the token values and every component updates instantly.
Yes. SuperCSS is framework-agnostic CSS. The Angular template is one port — the same class contracts work in React, Vue, Svelte, or plain HTML.
