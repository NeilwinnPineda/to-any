SuperCSS is a token-driven CSS design system. Every component consumes --sc-* custom properties so themes, palettes, and density scales mutate the visual output without touching markup.
Set data-theme, data-palette, data-mode, data-density, or data-radius on the html element. The mixer layer rewrites the token values and every component updates instantly.
